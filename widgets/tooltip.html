content: 表示させるcontent を指定

items: 何の属性をtooltip として表示するか指定, default: [title]

※Note: content を指定しなければならない。 



<div class="preview">
  <img src="http://placehold.jp/100x100.png" alt="alt attribute is tooltip" class="tooltip-items" width="100" height="100">
</div>





$('.tooltip-items').tooltip({
  items: "img[alt]",
  content: 'This is img[alt] attribute.'
});

position: tooltipを表示する位置を指定, default: { my: "left top+15", at: "left bottom", collision: "flipfit" }



<label for="tooltip-position">changed position:</label>
<input type="number" id="tooltip-position" title="this tooltip changed position.">





$('#tooltip-position').tooltip({
  position: {
    at: 'right center',
    my: 'left+15 center',
  }
});

tooltipClass: tooltipに適用するclassを指定



<label for="tooltip-tooltipClass">tooltipClass:</label>
<input type="number" id="tooltip-tooltipClass" title="this tooltip use tooltipClass, then changed color to red.">





.danger {
  color: #c33;
}





$('#tooltip-tooltipClass').tooltip({
  tooltipClass: 'danger'
});

track: tooltipが表示されている間、マウスに付いてくる



<label for="tooltip-track">track:</label>
<input type="number" id="tooltip-track" title="this tooltip is tracked.">





$('#tooltip-track').tooltip({
  track: true
});

Events

close( event, ui ): focusout or mouseleave をしたら、イベント発生

msg:


<div class="div-tooltip-close">msg:</div>


<label for="tooltip-close">close:</label>
<input type="number" id="tooltip-close" title="if focusout or mouseleave, trigger.">





$('#tooltip-close').tooltip({
  close: function( event, ui ) {
    $('.div-tooltip-close').text(ui.tooltip.text());
  }
});

open( event, ui ): focusin or mouseoverをしたら、イベント発生

msg:


<div class="div-tooltip-open">msg:</div>


<label for="tooltip-open">open:</label>
<input type="number" id="tooltip-open" title="if &quot;focusin&quot; or &quot;mouseover&quot;, trigger.">





$('#tooltip-open').tooltip({
  open: function( event, ui ) {
    $('.div-tooltip-open').text(ui.tooltip.text());
  },
  close: function( event, ui ) {
    $('.div-tooltip-open').text('msg:');
  }
});
