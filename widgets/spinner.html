Methods

value()



<label for="spinner-methods-value">input: </label>
<input id="spinner-methods-value" name="input" value="0"></input>


<button class="btn-spinner-methods-value">Get</button>
<input type="number" name="output" class="input-spinner-methods-value">





$('#spinner-methods-value').spinner();


$('.btn-spinner-methods-value')
.click( function() {
  var val = $('#spinner-methods-value').spinner('value');
  $('.input-spinner-methods-value').val(val);
});

Events

change( event, ui ): 値を変えてフォーカスを外したらイベント発生



<label for="spinner-change">input: </label>
<input id="spinner-change" name="input" value="0"></input>


<input type="number" name="output" class="input-spinner-change">





$('#spinner-change').spinner({
  change: function(event, ui) {
    var val = $(this).spinner('value');
    $('.input-spinner-change').val(val);
  }
});
