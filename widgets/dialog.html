Options

autoOpen

false にしないとだdialogの本文が最初から表示される。

The dialog window can be moved, resized and closed with the 'x' icon. 



<button class="btn-dialog-autoOpen">Open!</button>


<div class="dialog-autoOpen" title="autoOpen">
  <p>
    false にしないとだdialogの本文が最初から表示される。<br>
    <br>
    The dialog window can be moved, resized and closed with the 'x' icon.
  </p>
</div>





$('.dialog-autoOpen').dialog({
  autoOpen: false
});


$('.btn-dialog-autoOpen').click(function(){
  $('.dialog-autoOpen').dialog('open');
});

buttons

ボタンを追加したり、ボタンのテキストや機能を変更。 



<button class="btn-dialog-buttons">Open!</button>


<div class="dialog-buttons" title="buttons">
  <p>
    ボタンを追加したり、ボタンのテキストや機能を変更。
  </p>
</div>





$('.dialog-buttons').dialog({
  autoOpen: false,
  buttons: [
    {
      text: "OK!",
      icons: {
        primary: "ui-icon-heart"
      },
      click: function() {
        $( this ).dialog( "close" );
      }
    },
    {
      text: "Close!",
      click: function() {
        $( this ).dialog( "close" );
      }
    },
  ]
});


$('.btn-dialog-buttons').click(function(){
  $('.dialog-buttons').dialog('open');
});

position

position:
基点となる要素（of）
の位置（at）と、
ダイアログの位置（my）とが接するように決める。 



<button class="btn-dialog-position">Open!</button>


<div class="dialog-position" title="position">
  <p>
    position:<br>
    基点となる要素（of）<br>
    の位置（at）と、<br>
    ダイアログの位置（my）とが接するように決める。
  </p>
</div>





$('.dialog-position').dialog({
  autoOpen: false,
  position: {
    of: '.btn-dialog-position',
    at: 'center bottom',
    my: 'left top'
  }
});


$('.btn-dialog-position').click(function(){
  $('.dialog-position').dialog('open');
});

size.width: size.height:

resize: リサイズすると、イベント発生。

<button class="btn-dialog-resize">Open!</button>
<div class="dialog-resize" title="resize">
  <p>

  </p>
</div>





$('.dialog-resize').dialog({
  autoOpen: false,
  resize: function( event, ui ) {
    $('.div-dialog-resize').text('width: ' + Math.floor(ui.size.width) + ' height: ' + Math.floor(ui.size.height));
  }
});

  $('.dialog-resize').dialog('open');
});
