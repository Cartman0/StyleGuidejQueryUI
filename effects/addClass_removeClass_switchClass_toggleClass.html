addClass

addClass 公式DEMO addClass Options・Events詳細

.addClass( className [, duration ] [, easing ] [, complete ] )

easing でアニメーション(effect)を付与しながら、classを追加できる。



<div class="box div-addClass"></div>





.effeccts-addClass-box {
  width:100px;
  height:100px;
  border: 1px solid #333;
}


.effeccts-addClass__big-danger {
  background-color: #c33;
  width: 200px;
  height: 200px;
}





$('.effeccts-addClass-box').click( function() {
  $(this).addClass('effeccts-addClass__big-danger', 1000, "easeOutBounce" );
});

switchClass

switchClass 公式DEMO switchClass Options・Events詳細

.switchClass( className [, duration ] [, easing ] [, complete ] )

addClass, removeClassを指定。



<div class="effects-switch-box effects-switchClass-info"></div>





.effects-switchClass-info {
  background-color: #37c;
  width: 100px;
  height: 100px;
}


.effects-switchClass-danger {
  background-color: #c33;
  width: 200px;
  height: 200px;
}





$('.effects-switch-box').click(function() {
  $('.effects-switchClass-info').switchClass('effects-switchClass-info',  'effects-switchClass-danger', 1000, 'easeInBack');
  $('.effects-switchClass-danger').switchClass('effects-switchClass-danger', 'effects-switchClass-info', 1000, 'easeInBack');
});
